# Project Short Descriptions & Links

## 1) Mio
A mobile social platform built with React Native and Expo to connect users through shared entertainment interests.
The app helps people discover others who enjoy similar TV series and movies, making it easier to start conversations.
It focuses on a clean user experience, smooth navigation, and practical social discovery features for daily use.
Overall, Mio demonstrates cross-platform mobile development, real-world product thinking, and community-focused design.

## 2) Shop Circuit
Shop Circuit is a full ecommerce platform designed for modern online shopping with a dedicated admin panel.
It supports structured product browsing, order handling, and secure payment processing through SSLCommerz integration.
The frontend was developed with React, React Router, Tailwind CSS, and Firebase SDK for a fast and responsive UX.
The backend uses Node.js and Express to manage core business logic and support scalable API-driven workflows.

- Live: https://www.shopcircuit.me/

## 3) Bloodshot
Bloodshot is a fast-paced action game developed in Unity where every bullet fired costs player health.
This core mechanic creates a high-risk combat loop that forces strategic decision-making in every encounter.
Players control a goblin character and survive against waves of enemies through movement, timing, and resource control.
The project highlights gameplay system design, balancing mechanics, and end-to-end game development in C#.

- GitHub: https://github.com/AuvroIslam/BloodShot
- Live: https://auvro-islam.itch.io/bloodshot
- Video: https://youtu.be/bQRioChJImM?si=CB3LfQxl1ddhc8Q5

## 4) FrostByte Logistics Platform
FrostByte Logistics Platform is a cloud-native system engineered using a microservices architecture and DevOps workflows.

## 5) Waste Recognition Model
Waste Recognition Model is a deep learning project focused on classifying waste into meaningful disposal categories.
The model was developed using Fast.ai and integrated with Hugging Face for accessible deployment and experimentation.
To make it practical for end users, a web interface was built using HTML, CSS, and JavaScript.
This project demonstrates applied AI, model deployment workflow, and user-facing integration of ML solutions.

- GitHub: https://github.com/AuvroIslam/wasteRecognizer
- Live: https://auvroislam.github.io/wasteRecognizer/

## 6) GDP vs. Olympic Performance
This project explores how a country’s GDP may relate to its Olympic performance across different datasets.
Insights were presented through Tableau dashboards to highlight trends, patterns, and comparative country-level outcomes.
A React-based website was developed to publish and organize the visual stories in an interactive format.
It demonstrates end-to-end data storytelling, combining analysis, visualization, and frontend presentation.

- GitHub: https://github.com/AuvroIslam/Olympic_vs_Gdp
- Live: https://olympic-vs-gdp-website.vercel.app

## 7) 3Knot3
3Knot3 is a top-down 3D action game inspired by the seven Bir Sreshtho of Bangladesh’s 1971 Liberation War.
The game was built in Unity with C#, with each character and level reflecting heroic themes and historical inspiration.
It features ability-driven gameplay and environment design aimed at creating both action and narrative immersion.
The project represents creative game direction, technical implementation, and culturally meaningful storytelling.

- Live: https://studio-71.itch.io/echoes-of-71
